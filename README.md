### 基于vue的京东首页
### 技术栈
-vue
-vue-cli
-vuex
-vue.router
-less
### 下载本项目
`git clone 'git@github.com/hongjun-6722/jd_vue_text'`
## 项目安装
```
npm install
```
### 项目运行
```
npm run serve
```
### 项目构建
```
npm run build
```
